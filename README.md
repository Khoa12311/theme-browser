# 🎧 Hatsune Miku Cyber Theme
![Theme](https://img.shields.io/badge/theme-miku-cyber-blue)

A clean, dark cyber-style theme inspired by Hatsune Miku 💙

## ✨ Features
- Neon cyan accent
- Dark minimal UI
- Anime background (Miku)
- Optimized for readability

## 📸 Preview
![Preview](preview.png)

## 🚀 Installation

1. Download this repo as ZIP
2. Extract it
3. Open browser and go to:
   coccoc://extensions
4. Enable Developer Mode
5. Click "Load unpacked"
6. Select this folder

## 📦 Files
- `manifest.json` → theme config
- `images/bg.png` → background
- `images/icon.png` → icon
